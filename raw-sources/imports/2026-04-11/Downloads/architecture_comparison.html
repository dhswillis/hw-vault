3. The Local Agent Solution (Stop Burning Credits)
Proposed: Claude Code + GitHub Repo as Your Local Agent

Claude Code (the CLI tool) runs in your Mac terminal. Unlike Cowork, it can:

  • Auto-read CLAUDE.md — Put project context in a CLAUDE.md file at the repo root. Claude Code reads it automatically every session. No re-explaining.
  • Run SSH directly — No sandbox. scp, ssh, git push all work natively from your terminal.
  • Execute code — Run backtests, Python scripts, npm commands directly. No container restrictions.
  • Use your Max subscription — Same credits you're already paying for. No extra API costs.
  • Stay in project context — Point it at the trading-system/ folder and it understands the whole repo.
