# Batch Runner — Cluster Summary

> Six operational documents for the QC batch backtesting system: delivery summary, implementation notes, checklist, README, index, and examples.

## Contents

The batch runner is the nightly parameter sweep infrastructure on the VPS that feeds all mining reports. These six docs cover: how it was built (DELIVERY_SUMMARY, IMPLEMENTATION_NOTES), how to use it (README, CHECKLIST, EXAMPLES), and what's in it (INDEX of all batch configs).

See [[wiki/summaries/tempo-batch-cheatsheet|Tempo Batch Cheatsheet]] for the user-facing quick reference.

## Status / caveats

Operational infrastructure docs. The batch runner is actively running nightly on the VPS.
